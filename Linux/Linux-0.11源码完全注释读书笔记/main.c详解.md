# main文件详解

## 工作流程

1. 一系列初始化：根设备号、内存分区、中断向量、块设备、字符设备、tty(串行通信接口)、开机时间、进程调度、缓冲管理、硬盘、软驱的初始化；
2. 开中断；
3. 使用当前程序上下文创建任务0，并将任务0切换到用户态；
4. 创建init进程(任务1)；
5. 任务0和任务1分时执行，任务0执行`pause()`,任务1用于创建shell.
